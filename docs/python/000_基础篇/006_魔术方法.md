# 魔术方法

| method     | desc                                                                      |
| ---------- | ------------------------------------------------------------------------- |
| `__new__`  | 初始化对象 `__init__` 之前用于构造对象，返回实例给 `__init__` 中的 `self` |
| `__init__` | 在创建对象时自动调用的方法                                                |
| `__del__`  | 析构函数，在垃圾回收时执行                                                |
| 控制属性读写 |
| 算数运算符 |
| `__and__`  | 重写 `&` 的作用                                                           |
| `__or__`   | 重写 `\|` 的作用 |
| ... | 
| 比较运算符 |
| `__eq__`   | `==`                                                                      |
| `__ne__`   | `!=`                                                                      |
| `__le__`   | `<=`                                                                      |
| ... |
