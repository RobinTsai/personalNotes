# langChain

- [LangChain 中文网](https://www.langchain.com.cn/getting_started/getting_started)
- [LangChain 中文入门教程](https://www.bookstack.cn/read/LangChain-Chinese-Getting-Started-Guide/langchain.md)
- LangChain 是个框架，是个用于开发**由语言模型驱动的应用程序**的框架。安装使用 `pip install langchain`
- LangChain 通常与模型提供程序、数据存储、API 等集成，如要集成 OpenAI，安装 SDK `pip install openai` 并设置 OPENAI_API_KEY 环境变量
- LLM, Large Language Model，大语言模型
- 向量数据库，是一种数据相关性搜索的库，原理是先将数据向量化，
